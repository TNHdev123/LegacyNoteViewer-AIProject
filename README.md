# Legacy Note Viewer - AI Project
View Legacy iOS Apple Official Note App Notes on Your Bowser

# Download and Use
Download the LNV.html of this repository
Then use the browser or HTML viewer to run the file

# CSV Support
You can upload \var\mobile\Library\Notes\note.sqlite
and You can also upload the converted .csv files.
